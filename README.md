# Assuromieux Paris — Refonte

Ce dossier est la base de travail de la refonte du site Assuromieux Paris.

## Règles de sécurité

- Ne jamais modifier directement le fichier `source/index-production-reference.html`.
- Travailler dans une branche ou sur une copie locale.
- Tester chaque évolution avant déploiement Netlify.
- Conserver une copie du dernier site de production.

## Fichiers principaux

- `index.html` : copie de travail de la landing actuelle.
- `source/index-production-reference.html` : référence visuelle et technique à préserver.
- `docs/CAHIER-DES-CHARGES.md` : vision stratégique et fonctionnelle.
- `docs/ARCHITECTURE-SITE.md` : arborescence cible.
- `docs/TODO.md` : plan de travail par phases.
- `docs/PROMPT-CODEX.md` : consigne initiale à donner à Codex.
- `content/INVENTAIRE-CONTENUS.md` : contenus à rassembler.
- `seo/PLAN-SEO.md` : architecture SEO initiale.

## Éléments encore à ajouter manuellement

- Images originales et visuels métiers dans `assets/images/`.
- Mentions légales définitives.
