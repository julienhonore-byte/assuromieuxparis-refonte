# Prompt initial pour Codex

Tu travailles sur la refonte complète du site Assuromieux Paris.

Le dossier contient une copie du fichier HTML actuellement utilisé en production. Le fichier `source/index-production-reference.html` constitue la référence obligatoire pour l'identité visuelle, les couleurs, les composants, le logo, les formulaires et les comportements existants.

Commence par analyser intégralement le projet. Ne lance pas encore la refonte complète.

Ta première mission est uniquement de :

1. lire tous les fichiers du dossier ;
2. analyser le HTML, le CSS et le JavaScript existants ;
3. identifier les couleurs, typographies, espacements, composants, boutons, cartes, header, footer, formulaires et règles responsive ;
4. identifier les erreurs, doublons, fragilités et limites techniques ;
5. proposer une architecture de projet propre ;
6. créer `AUDIT-TECHNIQUE.md` ;
7. créer `PLAN-DE-REFONTE.md` ;
8. compléter `TODO.md` avec des tâches précises et ordonnées ;
9. ne modifier aucun fichier de production avant validation du plan.

Objectif final du projet : transformer la landing actuelle en un site premium, multi-pages, rapide, responsive, accessible et optimisé SEO, sans perdre l'identité graphique existante.

Le site doit positionner Assuromieux Paris comme cabinet de courtage et de conseil en assurances entreprises, avec une expertise forte en transport, logistique, flotte, TPM, RC Pro, BTP, santé, prévoyance et protection du dirigeant.

Le rendu final doit évoquer un cabinet de conseil premium, pas un comparateur d'assurances.
